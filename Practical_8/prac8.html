Introduction to Git

Git is a distributed version control system that helps manage and track changes in source code during software development.

Basic Git Commands

Here are some fundamental Git commands:

            git init
            git add
            git commit
            git status
            git log

Branching in Git

Git allows branching for parallel development. Common branch commands include:

            git branch
            git checkout
            git merge

Working with Remotes

Git supports collaboration by using remotes. Key remote commands include:

            git remote add
            git pull
            git push
